Carte matériaux

Carte matériaux is a free and open-source for editing a type Ashby map which can be used to choose engineering materials.
I used it to show the mechanical properties I collected from palm trees (ERC PROJETC-DOPAMICS) across different types of materials.

You need to run this code with the following packages : 

library(readxl)
library(dplyr)
library(ggplot2)
library(scales)
library(tidyr)
library(tidyverse)
library(patchwork)

Every section are detailled and separated with the #

Use the database "Data and information for engineering materials.xlsx"

Several lines should be modified to adapt the repository, color, legend, ...
